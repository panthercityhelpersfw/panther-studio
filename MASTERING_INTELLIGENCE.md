# Mastering Intelligence

Mastering intelligence evaluates the rendered master mix and project master settings.

## Metrics

- Peak dB
- RMS dB
- Loudness approximation
- Crest factor
- Clipping percentage
- Tonal balance ratios
- Stereo width
- Mono-below setting

## Targets

The recommendation layer maps user-facing targets onto existing mastering targets:

- Spotify / Apple Music: streaming
- YouTube: youtube
- TikTok: tiktok
- Podcast: quiet-podcast
- Loud demo: loud
- Dynamic audiophile: clean-pop

## Safety

Export warnings are evidence-backed. Limiter recommendations are triggered by measured peak/clipping risk, not generic advice.
