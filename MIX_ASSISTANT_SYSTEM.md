# Mix Assistant System

The mix assistant uses rendered stems and measurable audio data.

## Existing Mix Tools

- `analyzeMix`
- `applyMixFix`
- `autoGainStage`
- `levelRideClip`
- bus routing

## Studio Intelligence Additions

- Vocal masking risk
- Kick/bass overlap warning
- Mix clarity score
- Vocal presence score
- Beat/vocal compatibility score
- Smart FX preset recommendations

## One-Click Fixes

- Bring Vocal Forward
- De-Harsh Vocal
- Auto gain staging
- Level riding
- Limiter protection
- Mono/width translation tightening

All fixes mutate project state through existing undoable store paths.
